The fine dinner
===============

Pyweek 35 individual entry by Alexander Izmailov.

How to play
-----------

Install the lastest pyglet:

    pip install -r requirements.txt

or

    pip install pyglet==2.0.5

Launch:

    ./run_game.py

Controls
--------

    WSAD - move
    O - review goals
    Esc - exit

Cheatcodes
----------
    Ctrl+N - win this level
